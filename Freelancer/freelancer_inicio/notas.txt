Responsive Web Design y Media Queries
    Es un enfoque que nos dice que nuestros diseños deberian 
    adaptarse a las interacciones del usuario y la resolucion que utilizan.

    Los sitios web deberian adaptarce a celulares, tablets, laptops, computadoras
    de escrotorio, televisores.

    Esto se logra utilizando Media Queries.
    @media (min-whith:768px){}
    @media (min-whith:992px){}

CSS Box Model o Modelo de Caja
    Es el tamaño de los que se muestra en pantalla donde esta limitado por 4
    cosas :
        1. Tamaño del contenido.
        2. Tamaño de relleno (padding).
        3. Tamaño de borde
        4. El margen
    Se recomienda utilizar en la etiqueta html dentro del css el siguiente elemento:
        html {
            box-sizing: border-box;
        }
        *, *:before, *:after {
            box-sizing: inherit;
        }
    Con estas lineas no modificara el tamaño de los elementos al agregar o quitar pading.
